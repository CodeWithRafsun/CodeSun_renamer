# Changelog

All notable changes to CodeSun Renamer are documented here.

## [v2.0.1] - 2026

### Added

- New CodeSun branded CLI experience
- Startup brand banner system
- Success installation banner
- Advanced TUI style interface
- Human friendly installation messages
- Loading animation during setup
- Command help system

### New Rename Features

- Prefix + Suffix rename
- Filename case converter
    - lowercase
    - UPPERCASE
    - Title Case
- Remove specific text from filename
- Change file extension

### Improved

- Better installation flow
- Improved Termux/Linux support
- Cleaner project structure
- Better user interaction
- Improved error messages
- Safer rename preview system

### Existing Features

- Batch file rename
- Find and replace rename
- Date based rename
- Undo last rename

### Technical

- No external Python dependency
- Pure Python based
- Lightweight CLI utility
- Works on Linux and Termux


## [v1.0]

Initial release.

- Basic batch renaming
- Find & replace
- Date rename
- Undo support
